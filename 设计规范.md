# 中亚新闻地图 — 设计规范

**版本**：v0.1
**最后更新**：2026-04-30

---

## 1. 设计理念

**关键词**：严肃、探索、地域感

- 深色背景突出地图与点位，避免大面积白底带来的"工具感"
- 信息层级清晰：地图为主，新闻为辅，弹窗为焦点
- 配色融入中亚地域感（突厥蓝 + 沙色暖调）

---

## 7. 组件样式

### 7.1 地图点位（Marker）

**默认状态**
- 形状：圆点
- 大小：12px
- 颜色：按国家配色
- 边框：2px 白色（提升地图上的可见度）
- 阴影：`0 2px 6px rgba(0,0,0,0.4)`

**悬浮状态**
- 大小：16px
- 边框：2px 白色
- 外发光：8px 同色半透明

**选中状态**
- 大小：18px
- 内圈：白色 4px
- 外圈呼吸动画

**聚合点位（多条新闻聚集）**
- 形状：圆形带数字
- 大小：随聚合数量 24px-40px
- 背景：`rgba(59, 158, 255, 0.85)`
- 文字：白色 12px 加粗

### 7.2 新闻弹窗（InfoWindow）


### 7.3 国家标签

- 圆角：sm
- 内边距：4px 8px
- 字号：11px
- 背景：对应国家色 20% 透明度
- 文字：对应国家色 100%

示例：`KZ` `UZ` `KG` `TJ` `TM`

### 7.4 按钮

**主按钮**
```css
background: #3B9EFF;
color: #FFFFFF;
padding: 8px 16px;
border-radius: 8px;
font-weight: 500;
transition: background 0.15s;
```

悬浮：`#5BB0FF`
按下：`#2A8DEF`

**次按钮**
```css
background: transparent;
color: #E8EEF5;
border: 1px solid #2A3548;
```

### 7.5 侧边新闻列表

- 宽度：360px（屏宽 < 1280 时折叠为浮层）
- 滚动容器：单条新闻 padding: lg
- 单条新闻：上下 1px Border 分隔
- hover：背景变 `#1F2A3E`，左侧出现 3px 国家色竖条
- 选中（与地图联动）：背景 `#243248`，左侧 3px Sand 暖色竖条

### 7.6 顶部状态栏

```
[Logo: 中亚新闻地图]   [🔄 更新于 5 分钟前]   [国家筛选]   [关于]
```

- 高度：56px
- 背景：Surface 95% + backdrop-filter blur(12px)
- 底部：1px Border

## 8. 交互规范

### 8.1 加载状态
- 页面初次加载：地图区域骨架屏 + 中央转圈
- 数据刷新：右上角 toast「正在更新…」

### 8.2 空状态
当筛选后无新闻：
```
[图标]
该地区最近 7 天暂无新闻
[查看全部国家]
```

### 8.3 错误状态
- API 失败：顶部红色提示条「数据加载失败，[重试]」
- 高德加载失败：fallback 提示「地图加载失败，请检查网络」

### 8.4 动效
- 点位出现：缩放进入 200ms cubic-bezier(0.4, 0, 0.2, 1)
- 弹窗：淡入 + 上移 4px，150ms
- 列表项 hover：背景变化 100ms
- 整体动效原则：克制、快速、不喧宾夺主
