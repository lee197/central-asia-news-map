# 00 · 执行总览

**项目**：中亚新闻地图 MVP
**作者**：Lee Qi
**最后更新**：2026-04-30

---

## 你将得到什么

一个能在浏览器里打开的网站：地图上显示中亚五国的新闻点位，点击查看中文标题和摘要，跳转外站看原文。

部署完成后访问方式：
- 你打开 VPN
- 访问 `https://你的网站.vercel.app`
- 看到地图和新闻点

---

## 整体架构（再次确认）

```
                ┌──────────────────────────┐
                │  Vercel（前端托管）       │
                │  index.html + 高德地图    │
                └────────────┬─────────────┘
                             │ fetch /api/news
                             ↓
                ┌──────────────────────────┐
                │  Vultr 东京 VPS（$6/月） │
                │  ┌──────────────────┐   │
                │  │ FastAPI 后端服务  │   │
                │  │ SQLite 数据库     │   │
                │  │ 定时抓取脚本      │   │
                │  └──────────────────┘   │
                └────────┬────────┬────────┘
                         │        │
                         ↓        ↓
                    GDELT/RSS   Claude API
                    (拉新闻)    (翻译+定位)
```

**所有服务都在国外**，你访问需要 VPN。

## 重要约定

### 关于代码

- 所有代码我会写好放在文档里
- 你只需要**复制粘贴**到服务器上运行
- 如果运行报错，**完整把错误信息发给我**

### 关于报错

新手最重要的能力是：**遇到错误不要慌，把错误信息原样发给我**。

不要说"它报错了"，要说"它报错了，错误信息是：[完整粘贴]"。
错误信息里的英文也不用看懂，原样给我就行。

### 关于操作

每个文档每一步都有：
- ✅ **要做什么**
- 📋 **具体命令**（你复制就行）
- 🎯 **完成标志**（确认这步做对了）
- ⚠️ **可能踩的坑**

### 关于 VPN

整个开发过程你的电脑都需要开着 VPN，否则：
- 连不上 Vultr 服务器
- 调不了 Claude API
- 注册 Vercel 也不行
