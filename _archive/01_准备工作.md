# 01 · 准备工作

**预计耗时**：2-3 小时
**前置条件**：海外信用卡 + VPN

---

## 这一步要做什么

我们要在动手写代码之前，把所有账号、密钥、工具都准备好。这一步看起来繁琐，但**做完后面会非常顺**。

清单：
1. ✅ 装 VS Code（写代码用）
2. ✅ 装 iTerm2（更好用的终端）
3. ✅ 注册 Vultr 并购买云服务器
4. ✅ 注册 Anthropic 拿 Claude API key
5. ✅ 注册高德开放平台拿地图 key
6. ✅ 学会用 SSH 连服务器

---

## 步骤 1：装 VS Code

**为什么要装**：写代码、改文件、看错误日志都用它。比 Mac 自带的文本编辑器强 10 倍。

### 操作

1. 打开浏览器访问：https://code.visualstudio.com/
2. 点击大蓝按钮 "Download for Mac"
3. 下载完成后双击 `.zip` 文件解压
4. 把解压出来的 "Visual Studio Code" 拖到"应用程序"文件夹
5. 打开 VS Code

🎯 **完成标志**：能打开 VS Code，看到欢迎页。

---

## 步骤 2：装 iTerm2（终端工具）

**为什么要装**：Mac 自带终端能用，但 iTerm2 更好用，颜色清晰，操作方便。

### 操作

1. 浏览器访问：https://iterm2.com/
2. 点击 "Download" 下载
3. 解压后拖到"应用程序"文件夹
4. 打开 iTerm2，看到一个黑色窗口和命令提示符（类似 `leeqi@Mac ~ %`）

🎯 **完成标志**：iTerm2 能打开，看到命令提示符。

### 怎么用终端

后续文档里所有以 `$` 开头或者写在代码块里的命令，都是要你在终端里粘贴执行的。

例如：
```bash
echo "hello"
```
你只需要复制 `echo "hello"`（不要复制 `$`），粘贴到 iTerm2，按回车。

**测试一下**，在 iTerm2 里粘贴这个命令并回车：
```bash
echo "hello world"
```
应该看到屏幕输出 `hello world`。

🎯 **完成标志**：你能成功执行上面这个命令。

## 步骤 3：注册 Vultr 并买云服务器

**为什么要 Vultr**：便宜（$6/月）、东京节点对中国友好、注册简单。

### 3.1 注册账号

1. ⚠️ **先确保 VPN 开着**
2. 访问：https://www.vultr.com/
3. 点击右上角 "Sign Up"
4. 填邮箱和密码注册
5. 邮箱验证

### 3.2 充值

Vultr 是预付费的，先充钱才能买服务器。

1. 登录后，左侧菜单点 "Billing"
2. 点 "Add Funds"
3. **充 $10**（够用两个月）
4. 选信用卡支付，填卡号

⚠️ **可能遇到的问题**：
- 信用卡被拒：换张卡试，或者用 PayPal
- 提示需要验证：按提示走，一般是信用卡 3D 验证

### 3.3 创建服务器

1. 左侧菜单点 "Products" → "Compute"
2. 右上角点 "Deploy +" 蓝色按钮
3. 按以下选择：

| 选项 | 选什么 |
|---|---|
| Choose Server | **Cloud Compute** |
| CPU & Storage Technology | **Regular Performance** |
| Server Location | **Tokyo** 🇯🇵 |
| Server Image | **Ubuntu 24.04 LTS x64** |
| Server Size | **$6/mo**（1 vCPU / 1 GB RAM / 25 GB SSD） |

下面继续：
- Add Auto Backups：**不勾**（省钱）
- Add DDoS Protection：**不勾**
- IPv6：**勾上**
- Server Hostname & Label：随便填，比如 `central-asia-news`

4. 滚到最下，点 "Deploy Now"
5. 等待 1-2 分钟，状态从 "Installing" 变成 "Running"

### 3.4 记下服务器信息

服务器创建好后，点击它进入详情页，记下这几个信息：

| 信息 | 你的值 |
|---|---|
| IP Address（IP 地址） | `xxx.xxx.xxx.xxx` |
| Username（用户名） | `root` |
| Password（密码） | 点眼睛图标显示，复制保存 |

⚠️ **务必保存好密码**！可以新建一个文本文件 `服务器信息.txt` 放在桌面，里面写：
```
Vultr 东京服务器
IP: xxx.xxx.xxx.xxx
用户: root
密码: xxxxxxxxxx
```

🎯 **完成标志**：你有一台 Running 状态的服务器，并记下了 IP 和密码。

## 步骤 4：测试连接服务器

现在试着连上你的服务器。

### 操作

1. 打开 iTerm2
2. 输入命令（把 `xxx.xxx.xxx.xxx` 换成你的真实 IP）：

```bash
ssh root@xxx.xxx.xxx.xxx
```

3. 第一次连接会问 "Are you sure you want to continue connecting (yes/no)?"
   输入 `yes` 回车

4. 它会问 password，**直接粘贴你的服务器密码**（粘贴时屏幕不显示任何字符是正常的，密码管理器在工作），按回车

5. 看到类似下面的提示就成功了：
```
Welcome to Ubuntu 24.04 LTS
root@central-asia-news:~#
```

⚠️ **可能遇到的问题**：
- 提示 "Connection refused" 或 "timeout"：等 2 分钟再试，服务器还在启动
- 密码不对：复制粘贴时多了空格，重新复制
- 一直卡住没反应：检查 VPN

### 退出连接

输入 `exit` 回车，回到本地。

🎯 **完成标志**：你能成功 SSH 连上服务器，看到 `root@xxx:~#` 提示符。

## 步骤 5：注册 Anthropic 拿 Claude API key

**为什么需要**：用来翻译新闻 + 抽取地点。

### 操作

1. ⚠️ **VPN 必须开着**
2. 访问：https://console.anthropic.com/
3. 点 "Sign Up" 注册（用 Google 账号最快）
4. 完成邮箱验证

### 充值

新用户通常有 $5 免费额度，但建议先充值确保能用：

1. 登录后，左侧 "Billing" → "Plans"
2. 点 "Add Credit"
3. **充 $10**（按你这个项目用量能撑 2-3 个月）
4. 信用卡支付

### 拿 API key

1. 左侧 "API Keys"
2. 点 "Create Key"
3. 名字随便填，比如 `central-asia-news`
4. 复制出来的 key（以 `sk-ant-` 开头）

⚠️ **这个 key 只会显示一次**！立刻复制保存到桌面的 `服务器信息.txt` 里：

```
Claude API Key: sk-ant-xxxxxxxxxxxx
```

🎯 **完成标志**：你有一个以 `sk-ant-` 开头的 API key，并妥善保存。

## 步骤 6：注册高德开放平台拿地图 key

**为什么需要**：网站上的地图。

### 操作

1. 访问：https://lbs.amap.com/（这个国内能直接访问，不需要 VPN）
2. 右上角 "登录/注册"
3. 用手机号注册（需要短信验证）
4. 完成实名认证（个人身份证认证，需要上传身份证照片）

### 申请应用

1. 登录后，进入 "控制台"：https://console.amap.com/
2. 左侧 "应用管理" → "我的应用"
3. 点 "创建新应用"
4. 应用名称：`中亚新闻地图`
5. 应用类型：选 "网站应用" 或类似
6. 点 "添加 Key"
7. 服务平台：**Web 端 (JS API)**
8. 提交

### 拿 Key

应用列表里能看到 Key（一串 32 位字母数字），复制保存：

```
高德 JS API Key: xxxxxxxxxxxxxxxxxxxxxxxx
```

⚠️ **注意**：高德 Web 端 Key 默认每天有 30000 次免费额度，对你这个 MVP 完全够用。

🎯 **完成标志**：你有一个高德 JS API key 并保存了。
